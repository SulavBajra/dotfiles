# Dotfiles

My personal configuration files for a Niri wayland window manager setup on Linux.

## Requirements

### Window Manager & Core
- **[Niri](https://niri-wm.github.io/niri/)** - Wayland window manager
- **[Waybar](https://github.com/alexays/waybar)** - Status bar
- **[Mako](https://github.com/emersion/mako)** - Notification daemon
- **[Fuzzel](https://codeberg.org/dnkl/fuzzel)** - Application launcher

### Applications
- **Dolphin** - File manager
- **Blueman** - Bluetooth support

## Setup

### Installation

```bash
git clone https://github.com/SulavBajra/dotfiles ~/.config/dotfiles
# Link configuration files as needed
```

### Recommended Applications

- **Ghostty** - Terminal emulator
- **Brave Browser** - Web browser

##  Theme & Font

- **Theme:** Tokyo Night Dark

## Keybinds

### Workspace Management
| Keybind | Action |
|---------|--------|
| <kbd>Super</kbd> + <kbd>1</kbd>/<kbd>2</kbd>/<kbd>3</kbd> | Switch workspace |
| <kbd>Super</kbd> + <kbd>W</kbd> | Overview |
| <kbd>Super</kbd> + <kbd>D</kbd> | Application launcher |
| <kbd>Alt</kbd> + <kbd>Ctrl</kbd> + <kbd>←</kbd>/<kbd>→</kbd> | Move window to next workspace |

### Applications
| Keybind | Action |
|---------|--------|
| <kbd>Super</kbd> + <kbd>T</kbd> | Terminal |
| <kbd>Super</kbd> + <kbd>B</kbd> | Browser |
| <kbd>Super</kbd> + <kbd>E</kbd> | File manager |

## Useful Commands

```bash
niri validate     # Validate Niri configuration
pokemon           # Display random Pokémon
```
